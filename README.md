# MC-SPEED-UP
Script that speed up your Minecraft changing the config


Features:
  - Speed up Minecraft (More fps)
  - Enable Full Gamma
  - Disable "Press E to open inventory"
  - Backup your Minecraft config
  - AutoUpdate checker
